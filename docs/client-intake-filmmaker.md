## Recomendacion Practica (Etapa Inicial)

Si todavia no quieres construir formulario propio, usa este flujo:

1. Enviar este cuestionario como Google Doc o Notion compartido.
2. Pedir que todos los archivos pesados se suban a una carpeta Drive con estructura clara.
3. Hacer una videollamada corta (30 min) para validar respuestas y cerrar dudas.
4. Convertir este brief en backlog de implementacion.

Cuando quieras, lo siguiente ideal es crear un formulario propio dentro de la web para profesionalizar aun mas el intake.
